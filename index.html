Activate Email OTP

TaskFlow sends a one-time code to the user's inbox on every signup.
This requires a free EmailJS account — takes about 3 minutes.

📖 How to get your keys (click to collapse)
1 Go to emailjs.com and create a free account
2 Add an Email Service (connect Gmail / Outlook) → copy the Service ID
3 Create an Email Template — use these variables in the body:
{{otp_code}}  {{to_name}}  {{to_email}}
Then copy the Template ID
4 Go to Account → API Keys → copy your Public Key
