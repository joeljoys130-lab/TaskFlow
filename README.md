# TaskFlow
A beautiful, smart todo list web application with native device notifications and Firebase Authentication.

## Features
- Task Management with Categories and Priorities
- Due Dates with specific exact Times
- Push Reminders (exactly 1 Hour before a task is due)
- OTP Authentication using Firebase & EmailJS
- Installable Web App (PWA) with Offline Support
- Mobile Responsive Design

## Setup
To run quickly, use the \`serve\` library or open the html directly:
\`\`\`bash
npx serve -l 3000
\`\`\`
